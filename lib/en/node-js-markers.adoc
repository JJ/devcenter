= Node.js Markers
Adding marker files to `.openshift/markers` will have the following effects:

[cols="1,3",options="header"]
|===
|Marker |Effect

|nodejs_version
|Will set the runtime node interpreter to that version, vor instance 0.9.1 or 4.3.1

|===
